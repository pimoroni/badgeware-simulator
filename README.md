# Badgeware Simulator

Simulator for the development of the Picovector graphics API and accompanying badgeware apps.

## Controls

| Key    | Action |
|--------|--------|
| H      | Home   |
| Left   | A      |
| Space  | B      |
| Right  | C      |
| Up     | Up     |
| Down   | Down   |
| H      | Screenshot |
| Escape | Hot Reload |

## Building

### Linux

Deps: `libxi-dev libxcursor-dev`

Running on a Pi: `LIBGL_ALWAYS_SOFTWARE=1`
